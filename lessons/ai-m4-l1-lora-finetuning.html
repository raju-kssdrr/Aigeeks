LoRA & QLoRA Fine-tuning

⏱ 55 min 📚 AI Engineering for Devs 🧩 Lesson 4.1
PEFT SetupDataset PreparationTraining LoopVRAM Optimization
Core Concepts

LoRA (Low-Rank Adaptation)

LoRA freezes the original model weights and injects small trainable rank-decomposition matrices into attention layers. Instead of updating all 7B+ parameters, only the rank-r matrices (r=8–64) are trained. Reduces trainable parameters by 10,000x while maintaining 95%+ of full fine-tune quality.

QLoRA

QLoRA combines 4-bit NormalFloat quantization of the base model with LoRA adapters in bf16. Enables fine-tuning a 70B model on a single 48GB GPU (or a 7B model on 8GB). The quantized base model is frozen; only LoRA adapters are updated. Uses double quantization and paged AdamW for further memory savings.

Dataset Preparation

Format data as instruction-response pairs (Alpaca format) or chat format (ShareGPT). Quality >> quantity — 1,000 high-quality examples outperform 100,000 noisy ones. Use datasets library for efficient loading. Apply chat template from the tokenizer. Typical split: 95% train, 5% eval.

VRAM Optimization

Key levers: (1) 4-bit quantization (QLoRA); (2) Gradient checkpointing — trade compute for memory; (3) Gradient accumulation — simulate larger batches; (4) Flash Attention 2 — 2–4x faster attention with lower memory; (5) batch_size=1 with accumulation=8–16 for stable training on consumer GPUs.

# QLoRA fine-tuning with HuggingFace PEFT + TRL
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig
from peft import LoraConfig, get_peft_model, TaskType
from trl import SFTTrainer, SFTConfig
from datasets import load_dataset
import torch

# 4-bit quantization config
bnb_config = BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_quant_type="nf4",
    bnb_4bit_compute_dtype=torch.bfloat16,
    bnb_4bit_use_double_quant=True,
)

model = AutoModelForCausalLM.from_pretrained(
    "meta-llama/Llama-3.2-3B-Instruct",
    quantization_config=bnb_config,
    device_map="auto",
)
tokenizer = AutoTokenizer.from_pretrained("meta-llama/Llama-3.2-3B-Instruct")

# LoRA config
lora_config = LoraConfig(
    r=16,                    # rank
    lora_alpha=32,           # scaling
    target_modules=["q_proj", "v_proj", "k_proj", "o_proj"],
    lora_dropout=0.05,
    task_type=TaskType.CAUSAL_LM,
)
model = get_peft_model(model, lora_config)
model.print_trainable_parameters()  # ~0.1% of total params

# Load dataset
dataset = load_dataset("json", data_files={"train": "data/train.jsonl"})

# Train
trainer = SFTTrainer(
    model=model,
    train_dataset=dataset["train"],
    args=SFTConfig(
        output_dir="./qlora-output",
        num_train_epochs=3,
        per_device_train_batch_size=1,
        gradient_accumulation_steps=8,
        learning_rate=2e-4,
        bf16=True,
        gradient_checkpointing=True,
    ),
)
trainer.train()
model.save_pretrained("./qlora-output/final")
