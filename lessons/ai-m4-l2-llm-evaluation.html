Module 4: Fine-tuning & Evaluation

LLM Evaluation Metrics

⏱ 40 min 📚 AI Engineering for Devs 🧩 Lesson 4.2
BLEU/ROUGEG-EvalHallucination ChecksBenchmark Suites
Core Concepts

BLEU & ROUGE

BLEU measures n-gram precision between generated and reference text — good for translation. ROUGE measures recall — good for summarization. Both require reference answers and correlate poorly with human judgment for open-ended generation. Use only when you have gold-standard references.

G-Eval & LLM-as-Judge

Use a powerful LLM (GPT-4, Claude Opus) to score outputs on dimensions: faithfulness, relevance, coherence, correctness (1–5 scale). Provide explicit scoring rubrics. LLM judges correlate better with human ratings than BLEU/ROUGE. DeepEval and Ragas automate this for RAG pipelines.

Hallucination Detection

Factual hallucinations occur when the model generates plausible-sounding but false statements. Detect with: (1) NLI-based entailment — does the response follow from the retrieved context? (2) Self-consistency — sample N responses and check agreement; (3) FactScore — decompose claims and verify each against a knowledge source.

Benchmark Suites

MMLU (57 academic subjects), HumanEval (code generation), HellaSwag (commonsense), TruthfulQA (factual accuracy), and MATH test general capabilities. For domain-specific use, build a custom eval set of 200–500 representative examples with human-written gold answers.

# LLM evaluation with DeepEval
from deepeval import evaluate
from deepeval.metrics import (
    AnswerRelevancyMetric,
    FaithfulnessMetric,
    ContextualRecallMetric,
    HallucinationMetric,
)
from deepeval.test_case import LLMTestCase

# Define a test case
test_case = LLMTestCase(
    input="What is RAG?",
    actual_output="RAG stands for Retrieval-Augmented Generation. It combines a retrieval system with an LLM to ground responses in factual documents.",
    expected_output="RAG is a technique that retrieves relevant documents and uses them as context for an LLM to generate grounded responses.",
    retrieval_context=["RAG (Retrieval-Augmented Generation) is an AI framework that enhances LLM responses by retrieving relevant information from external knowledge bases."]
)

# Run evaluation
metrics = [
    AnswerRelevancyMetric(threshold=0.7),
    FaithfulnessMetric(threshold=0.8),
    HallucinationMetric(threshold=0.3),
]

evaluate([test_case], metrics)
