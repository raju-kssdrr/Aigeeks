OpenAI/Anthropic API Integration

⏱ 35 min 📚 AI Engineering for Devs 🧩 Lesson 5.1

Core Concepts

SDK Setup & Auth

Use official SDKs — never raw HTTP. Set API keys via environment variables (ANTHROPIC_API_KEY, OPENAI_API_KEY), never hardcode. Both SDKs auto-retry on 429/5xx with exponential backoff. Configure timeout (30–60s) and max_retries (3) in the client constructor.

Streaming Responses

Stream LLM output token-by-token with stream=True. Dramatically improves perceived latency — users see text appear immediately. Use server-sent events (SSE) to push streamed tokens to the frontend. Always handle stream interruptions gracefully and track usage from the final chunk.

Rate Limit Handling

Implement token bucket rate limiting client-side. Track requests-per-minute and tokens-per-minute separately. Use a queue for burst traffic. Set up model fallbacks (GPT-4o → GPT-4o-mini, Opus → Sonnet) for degraded-but-functional behavior during quota exhaustion.

Cost Tracking

Log input tokens, output tokens, model, and timestamp for every request. Input costs ~3–5x less than output. Implement cost caps per user/request. Cache frequent identical prompts (Anthropic prompt caching gives 90% discount on cached tokens). Monitor cost per feature to identify expensive paths.

import anthropic
import os
from typing import Iterator

client = anthropic.Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"],
    timeout=30.0,
    max_retries=3,
)

# Streaming response
def stream_response(prompt: str) -> Iterator[str]:
    with client.messages.stream(
        model="claude-sonnet-4-6",
        max_tokens=2048,
        messages=[{"role": "user", "content": prompt}],
    ) as stream:
        for text in stream.text_stream:
            yield text
        
        # Access usage after stream completes
        usage = stream.get_final_message().usage
        log_cost(usage.input_tokens, usage.output_tokens, "claude-sonnet-4-6")

def log_cost(input_tokens: int, output_tokens: int, model: str):
    # Claude Sonnet pricing (approximate)
    PRICES = {"claude-sonnet-4-6": {"input": 3.0, "output": 15.0}}  # per 1M tokens
    p = PRICES.get(model, {"input": 3.0, "output": 15.0})
    cost = (input_tokens * p["input"] + output_tokens * p["output"]) / 1_000_000
    print(f"Cost: ${cost:.6f} | In: {input_tokens} | Out: {output_tokens}")

# Prompt caching for repeated context
def cached_analysis(document: str, question: str) -> str:
    response = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=1024,
        system=[{"type": "text", "text": document, "cache_control": {"type": "ephemeral"}}],
        messages=[{"role": "user", "content": question}]
    )
    return response.content[0].text
