Self-hosted LLMs (vLLM)

⏱ 50 min 📚 AI Engineering for Devs 🧩 Lesson 5.2
vLLM ServerQuantizationBatchingGPU Memory Management
Core Concepts

vLLM & PagedAttention

vLLM is the fastest open-source LLM inference engine. PagedAttention manages KV-cache in non-contiguous memory pages (like OS virtual memory), eliminating memory fragmentation. Enables 24x higher throughput than naive HuggingFace inference. Exposes an OpenAI-compatible REST API.

Quantization

INT4/INT8 quantization reduces model size and speeds inference at slight quality cost. GPTQ (post-training quantization) and AWQ (activation-aware) are production standards. AWQ outperforms GPTQ at 4-bit. For CPU inference, use GGUF format with llama.cpp. Ollama wraps llama.cpp for easy local deployment.

Continuous Batching

vLLM uses continuous batching — new requests join the batch as soon as a sequence finishes, rather than waiting for the entire batch to complete. Dramatically increases GPU utilization (from ~30% to 80%+) under real traffic patterns. Set --max-num-seqs and --max-num-batched-tokens based on your GPU VRAM.

GPU Memory Management

Rule of thumb: model parameters need ~2 bytes/param in fp16 (7B model ≈ 14GB). KV-cache takes additional memory proportional to context length × batch size. Use --gpu-memory-utilization 0.90 in vLLM. For multi-GPU: tensor parallelism (--tensor-parallel-size 2) splits a model across GPUs.

# Query vLLM via OpenAI-compatible client
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="not-needed",  # vLLM doesn't require auth by default
)

# Single inference
response = client.chat.completions.create(
    model="meta-llama/Llama-3.2-8B-Instruct",
    messages=[{"role": "user", "content": "Explain attention mechanisms."}],
    max_tokens=512,
    temperature=0.7,
)
print(response.choices[0].message.content)

# Batch inference (concurrent requests)
import asyncio
from openai import AsyncOpenAI

async_client = AsyncOpenAI(base_url="http://localhost:8000/v1", api_key="none")

async def batch_infer(prompts: list[str]) -> list[str]:
    tasks = [
        async_client.chat.completions.create(
            model="meta-llama/Llama-3.2-8B-Instruct",
            messages=[{"role": "user", "content": p}],
            max_tokens=256,
        )
        for p in prompts
    ]
    responses = await asyncio.gather(*tasks)
    return [r.choices[0].message.content for r in responses]

results = asyncio.run(batch_infer(["What is ML?", "What is AI?", "What is DL?"]))
