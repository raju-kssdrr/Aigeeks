Cost Optimization & Caching

⏱ 30 min 📚 AI Engineering for Devs 🧩 Lesson 5.3
Prompt CachingSemantic CacheToken CountingModel Routing
Core Concepts

Prompt Caching

Anthropic prompt caching stores processed prompt prefixes on their servers for 5 minutes (ephemeral) or longer. Cache hits cost 90% less than cache misses. Ideal for long system prompts, documents, or tool definitions that stay constant across requests. Mark cacheable content with cache_control: ephemeral.

Semantic Cache

Cache LLM responses by semantic similarity of the input query. On new request: embed the query, search cache for similar past queries (cosine > 0.95), return cached response if found. GPTCache and Redis with vector search implement this. Effective for FAQ-style apps with repetitive queries.

Token Counting

Count tokens before sending to avoid surprise costs and context overflow. Use tiktoken for OpenAI models, anthropic.count_tokens() for Claude. Budget rules: system prompt < 20% of context, user input < 40%, response budget = remainder. Truncate retrieved docs to fit budget.

Model Routing

Not every request needs the most powerful model. Route simple queries (classification, short Q&A) to fast/cheap models (Haiku, GPT-4o-mini). Route complex reasoning, code, or analysis to powerful models (Opus, GPT-4o). A routing classifier LLM call costs ~0.001¢ and can save 10–100x on expensive model calls.

import anthropic
from anthropic import Anthropic

client = Anthropic()

# Prompt caching — long document stays cached
def analyze_document(document: str, question: str) -> str:
    response = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=1024,
        system=[
            {
                "type": "text",
                "text": f"You are a document analyst. Here is the document:\n\n{document}",
                "cache_control": {"type": "ephemeral"}  # Cache the document prefix
            }
        ],
        messages=[{"role": "user", "content": question}]
    )
    usage = response.usage
    cache_savings = usage.cache_read_input_tokens * 0.9  # 90% discount on cached tokens
    print(f"Cache hits: {usage.cache_read_input_tokens} | Savings: ~{cache_savings} tokens")
    return response.content[0].text

# Model routing
def route_and_call(prompt: str) -> str:
    complexity_check = client.messages.create(
        model="claude-haiku-4-5-20251001",  # Cheapest model for routing
        max_tokens=10,
        system="Classify prompt complexity. Reply with only: simple | complex",
        messages=[{"role": "user", "content": prompt[:500]}]
    )
    complexity = complexity_check.content[0].text.strip()
    
    model = "claude-haiku-4-5-20251001" if complexity == "simple" else "claude-opus-4-7"
    response = client.messages.create(
        model=model, max_tokens=2048,
        messages=[{"role": "user", "content": prompt}]
    )
    print(f"Routed to: {model}")
    return response.content[0].text
