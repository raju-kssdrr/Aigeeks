Qubits, Superposition & Entanglement

⏱ 50 min 📚 Quantum Computing for Devs 🧩 Lesson 1.1

Core Concepts

Qubits vs Classical Bits

A classical bit is 0 or 1. A qubit is a superposition: |ψ⟩ = α|0⟩ + β|1⟩, where α and β are complex amplitudes with |α|² + |β|² = 1. The probabilities of measuring 0 or 1 are |α|² and |β|² respectively. Superposition is NOT both at once — it's a probability distribution that collapses on measurement.

Bloch Sphere

The Bloch sphere visualizes a single qubit state as a point on a unit sphere. The north pole = |0⟩, south pole = |1⟩, equator = equal superposition. Quantum gates rotate the state vector around axes of the sphere. Measuring collapses the state to the north or south pole.

Entanglement

Two qubits are entangled when their joint state cannot be written as a product of individual states. Measuring one qubit instantly determines the other regardless of distance. Bell state |Φ+⟩ = (|00⟩ + |11⟩)/√2 — measuring the first qubit as 0 guarantees the second is also 0. Enables quantum teleportation and superdense coding.

Quantum Measurement

Measurement is irreversible and destructive — it collapses the superposition to a definite value. You cannot copy a qubit (no-cloning theorem). To estimate probabilities, you must run the same circuit thousands of times and collect statistics. "Shots" = number of circuit repetitions.

# Qiskit basics
from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator

# Create a Bell state (maximally entangled)
qc = QuantumCircuit(2, 2)  # 2 qubits, 2 classical bits

qc.h(0)       # Hadamard: put qubit 0 in superposition
qc.cx(0, 1)   # CNOT: entangle qubit 0 and qubit 1
qc.measure([0, 1], [0, 1])  # Measure both qubits

print(qc.draw())

# Simulate
simulator = AerSimulator()
compiled = transpile(qc, simulator)
job = simulator.run(compiled, shots=1024)
counts = job.result().get_counts()
print("Counts:", counts)
# Expected: ~50% "00", ~50% "11" — never "01" or "10"
# This proves entanglement: measuring qubit 0 instantly determines qubit 1
