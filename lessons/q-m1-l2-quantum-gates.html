Quantum Gates & Circuits

Core Concepts

Single-Qubit Gates

Quantum gates are unitary matrices that rotate qubit state on the Bloch sphere. Pauli-X (NOT gate) flips |0⟩↔|1⟩. Pauli-Z flips the phase of |1⟩. Hadamard (H) creates superposition: H|0⟩ = (|0⟩+|1⟩)/√2. S and T gates rotate by π/2 and π/4. All single-qubit gates are reversible.

CNOT (Controlled-NOT)

The most important two-qubit gate. Flips the target qubit if and only if the control qubit is |1⟩. Together with single-qubit gates, CNOT forms a universal gate set — any quantum computation can be expressed using only these. Used to create entanglement.

Circuit Composition

Quantum circuits are sequences of gates applied left-to-right. Gates on different qubits can run in parallel. Circuit depth = longest path from input to output. Minimize depth to reduce decoherence on real hardware. Transpilation maps logical circuits to hardware-native gates (CX, SX, Rz on IBM devices).

Quantum Parallelism

Apply H to n qubits to create superposition of all 2ⁿ states simultaneously. A quantum function f applied to this superposition evaluates f on all inputs at once — but measurement only reveals ONE answer. The trick of quantum algorithms is to interfere amplitudes so the correct answer has high probability.

from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator
from qiskit.visualization import plot_histogram

# Single qubit gates demonstration
qc = QuantumCircuit(3, 3)

# Qubit 0: Hadamard then measure (50/50 superposition)
qc.h(0)

# Qubit 1: X gate (bit flip) then measure (always 1)
qc.x(1)

# Qubit 2: H + Z + H = X (phase kickback demonstration)
qc.h(2)
qc.z(2)
qc.h(2)

qc.barrier()
qc.measure([0, 1, 2], [0, 1, 2])

print(qc.draw(output="text"))

# Run simulation
sim = AerSimulator()
job = sim.run(transpile(qc, sim), shots=2048)
counts = job.result().get_counts()
print("Results:", counts)

# Toffoli gate (CCX) - AND gate equivalent
toffoli = QuantumCircuit(3, 1)
toffoli.x(0); toffoli.x(1)  # Set control qubits to |1⟩
toffoli.ccx(0, 1, 2)        # Flip target only if both controls are |1⟩
toffoli.measure(2, 0)
print("\nToffoli (AND gate):", toffoli.draw())
