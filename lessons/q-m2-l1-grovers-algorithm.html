Grover's Search Algorithm

⏱ 55 min 📚 Quantum Computing for Devs 🧩 Lesson 2.1
Oracle ConstructionAmplitude AmplificationQuadratic SpeedupApplications
Core Concepts

The Search Problem

Classical search of an unsorted N-item database requires O(N) queries on average. Grover's algorithm finds the target in O(√N) queries — a quadratic speedup. For N=1 million items, classical needs ~500,000 queries; Grover needs ~1,000. Not exponential, but significant for large N.

Oracle

The oracle is a quantum black box that flips the phase of the target state: |x⟩ → -|x⟩ if f(x)=1, else |x⟩. The oracle encodes the problem without revealing the answer. Constructing an efficient oracle is the hard part — for SAT problems, each clause maps to quantum gates.

Amplitude Amplification

Start with equal superposition. The oracle marks the target with a phase flip. The diffusion operator (2|s⟩⟨s|-I) reflects all amplitudes around the mean. Together: the target's amplitude grows, others shrink. After √N iterations, measuring gives the target with high probability.

Applications

Grover accelerates: database search, collision finding in cryptographic hash functions, solving NP problems with quadratic speedup, and constraint satisfaction. Relevant for breaking symmetric cryptography — AES-128 becomes AES-256-equivalent against a quantum adversary.

from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator
from qiskit.circuit.library import GroverOperator, PhaseOracle
import numpy as np

# Grover's search for a 2-qubit target
# Target: |11⟩ (binary 3)
n_qubits = 2
target = "11"

# Build oracle that marks |11⟩ with phase flip
oracle = QuantumCircuit(n_qubits)
oracle.cz(0, 1)  # For |11⟩: CZ flips phase when both qubits are |1⟩
oracle = oracle.to_gate(label="Oracle")

# Diffusion operator (inversion about average)
def diffusion(n):
    qc = QuantumCircuit(n)
    qc.h(range(n))
    qc.x(range(n))
    qc.h(n-1)
    qc.mcx(list(range(n-1)), n-1)  # Multi-controlled X
    qc.h(n-1)
    qc.x(range(n))
    qc.h(range(n))
    return qc.to_gate(label="Diffusion")

# Full Grover circuit
iterations = int(np.pi/4 * np.sqrt(2**n_qubits))  # Optimal: π/4 * √N
qc = QuantumCircuit(n_qubits, n_qubits)
qc.h(range(n_qubits))  # Initialize superposition

for _ in range(iterations):
    qc.append(oracle, range(n_qubits))
    qc.append(diffusion(n_qubits), range(n_qubits))

qc.measure(range(n_qubits), range(n_qubits))

sim = AerSimulator()
counts = sim.run(transpile(qc, sim), shots=1024).result().get_counts()
print("Search results:", counts)
# Expected: "11" appears with ~100% probability
