Shor's Factoring Algorithm

Core Concepts

The Factoring Problem

RSA encryption relies on the hardness of factoring large numbers. Classically, the best algorithm (GNFS) takes sub-exponential but superpolynomial time. Shor's algorithm factors N in O((log N)³) — polynomial time — making RSA theoretically breakable by a large enough quantum computer.

Quantum Fourier Transform (QFT)

The QFT is the quantum analogue of the DFT (Discrete Fourier Transform). Applied to a superposition, it transforms amplitudes from the computational basis to the frequency basis in O(n²) gates vs. classical FFT's O(n·2ⁿ). It is the core subroutine of Shor's and quantum phase estimation.

Period Finding

To factor N: pick random a < N, compute f(x) = aˣ mod N. This function is periodic with period r. If r is even and aʳ/² ≠ ±1 mod N, then gcd(aʳ/² ± 1, N) gives a non-trivial factor. The quantum part finds r exponentially faster using QFT-based phase estimation.

RSA Implications

A fault-tolerant quantum computer with ~4000 logical qubits (millions of physical qubits with error correction) could break RSA-2048 in hours. Current quantum computers have ~1000 noisy physical qubits — far short. NIST has standardized post-quantum cryptography (CRYSTALS-Kyber, CRYSTALS-Dilithium) in response.

# Simplified Shor's algorithm for small numbers (N=15)
from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator
from qiskit.circuit.library import QFT
from math import gcd

def shors_classical(N, a=7):
    """Classical simulation of the period-finding step."""
    # In reality this step is done quantumly
    x = 1
    for r in range(1, N):
        x = (x * a) % N
        if x == 1:
            # Found period r
            if r % 2 == 0:
                factor1 = gcd(a**(r//2) - 1, N)
                factor2 = gcd(a**(r//2) + 1, N)
                if factor1 not in [1, N] and factor2 not in [1, N]:
                    return factor1, factor2
    return None

N = 15
a = 7
factors = shors_classical(N, a)
print(f"Factors of {N}: {factors}")  # (3, 5)

# Quantum Fourier Transform circuit
n = 4
qft_circuit = QuantumCircuit(n)
qft = QFT(n, inverse=False)
qft_circuit.append(qft, range(n))
qft_circuit = qft_circuit.decompose()
print(f"\nQFT circuit depth: {qft_circuit.depth()}")
print(f"QFT gate count: {qft_circuit.size()}")
print(qft_circuit.draw())
