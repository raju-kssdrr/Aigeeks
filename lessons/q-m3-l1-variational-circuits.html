Core Concepts

Parameterized Quantum Circuits

Variational circuits (VQCs) contain rotation gates with trainable parameters θ: Rx(θ), Ry(θ), Rz(θ). A classical optimizer updates θ to minimize a cost function evaluated by quantum measurement. This hybrid classical-quantum loop is the foundation of NISQ-era quantum ML.

Ansatz Design

The ansatz is the circuit structure (entanglement pattern + gate sequence). Hardware-efficient ansatz uses only native gates and nearest-neighbor connectivity — shallower depth, less noise. Chemically-inspired ansatz (UCCSD) encodes physical structure for quantum chemistry. Expressibility vs trainability tradeoff.

Barren Plateaus

A critical failure mode: as circuit depth and qubit count grow, gradients become exponentially small (barren plateaus). The gradient vanishes so the optimizer cannot learn. Mitigations: layer-by-layer training, local cost functions, structured ansatz, gradient-free optimizers (COBYLA, SPSA).

Gradient Estimation

Parameter shift rule: ∂⟨C⟩/∂θ = [⟨C(θ+π/2)⟩ - ⟨C(θ-π/2)⟩] / 2. Compute gradients exactly using two circuit evaluations per parameter. More reliable than finite differences on noisy hardware. SPSA uses random simultaneous perturbations — fewer circuit calls at cost of noisier gradients.

import pennylane as qml
import numpy as np
from pennylane import numpy as pnp

dev = qml.device("default.qubit", wires=4)

@qml.qnode(dev)
def variational_circuit(params):
    # Layer 1: Single-qubit rotations
    for i in range(4):
        qml.RY(params[i], wires=i)
    # Entanglement layer
    qml.CNOT(wires=[0, 1])
    qml.CNOT(wires=[1, 2])
    qml.CNOT(wires=[2, 3])
    # Layer 2: Single-qubit rotations
    for i in range(4):
        qml.RY(params[i + 4], wires=i)
    return qml.expval(qml.PauliZ(0))  # Measure qubit 0

# Initialize random parameters
params = pnp.random.uniform(0, 2*np.pi, size=8, requires_grad=True)

# Compute gradient using parameter shift rule
grad_fn = qml.grad(variational_circuit)
gradients = grad_fn(params)
print("Gradients:", gradients)

# Optimization loop
optimizer = qml.AdamOptimizer(stepsize=0.1)
for step in range(50):
    params, cost = optimizer.step_and_cost(variational_circuit, params)
    if step % 10 == 0:
        print(f"Step {step}: Cost = {cost:.4f}")
