Core Concepts

QNN Architecture

A QNN has three parts: (1) Data encoding layer — map classical data to qubit states (angle encoding, amplitude encoding); (2) Variational circuit — parameterized gates acting as learnable weights; (3) Measurement layer — expectation values become the output. Each qubit measurement gives a scalar; combine for multi-class output.

Hybrid Classical-Quantum

QNNs are typically embedded inside classical neural networks. Classical layers preprocess data → QNN layer → classical post-processing. PennyLane qml.qnn.TorchLayer wraps a QNode as a PyTorch layer. Gradients flow through the quantum layer via parameter shift, enabling end-to-end backprop.

Data Encoding

Angle encoding: x_i → Ry(x_i) for each feature. Requires n qubits for n features — doesn't scale. Amplitude encoding: encode 2ⁿ values into n qubit amplitudes — exponentially compressed but hard to prepare. Re-uploading (data reuploading): repeat encoding through the circuit to increase expressibility.

Classical vs Quantum ML

For most practical ML tasks today, classical neural networks outperform QNNs. QNNs may offer advantage in: quantum data processing (molecular simulation), specific kernel methods, and possibly for exponentially large feature spaces. Current research is exploratory — treat with appropriate skepticism.

import pennylane as qml
from pennylane import numpy as np
import torch
import torch.nn as nn

n_qubits = 4
dev = qml.device("default.qubit", wires=n_qubits)

@qml.qnode(dev, interface="torch")
def quantum_layer(inputs, weights):
    # Data encoding
    for i in range(n_qubits):
        qml.RY(inputs[i], wires=i)
    # Variational layers
    qml.StronglyEntanglingLayers(weights, wires=range(n_qubits))
    # Measurements
    return [qml.expval(qml.PauliZ(i)) for i in range(n_qubits)]

# Wrap as PyTorch layer
weight_shapes = {"weights": (3, n_qubits, 3)}  # 3 layers, 4 qubits, 3 params each
qlayer = qml.qnn.TorchLayer(quantum_layer, weight_shapes)

# Hybrid model
class HybridQNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.classical_pre = nn.Linear(8, n_qubits)   # Classical pre-processing
        self.quantum = qlayer                           # Quantum layer
        self.classical_post = nn.Linear(n_qubits, 2)  # Classical post-processing
    
    def forward(self, x):
        x = torch.tanh(self.classical_pre(x)) * torch.pi  # Encode to [-π, π]
        x = self.quantum(x)                                 # Quantum forward pass
        return self.classical_post(x)

model = HybridQNN()
optimizer = torch.optim.Adam(model.parameters(), lr=0.01)
loss_fn = nn.CrossEntropyLoss()

# Dummy training step
x = torch.randn(16, 8)
y = torch.randint(0, 2, (16,))
logits = model(x)
loss = loss_fn(logits, y)
loss.backward()
optimizer.step()
print(f"Loss: {loss.item():.4f}")
