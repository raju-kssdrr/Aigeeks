Quantum Error Correction

⏱ 60 min 📚 Quantum Computing for Devs 🧩 Lesson 4.1

Core Concepts

Why Error Correction

Qubits decohere — they lose quantum information due to interaction with the environment. Decoherence timescales: superconducting qubits ~100μs, trapped ions ~1s. A deep circuit takes longer than coherence time → errors. Error correction encodes 1 logical qubit in many physical qubits, detecting and correcting errors without measuring the qubit state.

Bit-flip & Phase-flip Codes

Bit-flip code: |0⟩→|000⟩, |1⟩→|111⟩. Majority vote detects single bit flips. Phase-flip code: same idea in Hadamard basis. Shor code combines both to correct arbitrary single-qubit errors using 9 physical qubits per logical qubit. First complete quantum error correcting code.

Surface Code

The leading practical QEC scheme. Qubits arranged on a 2D lattice. Syndrome measurements (parity checks on neighboring qubits) detect errors without reading qubit values. Threshold: ~1% physical error rate — if physical gates are better than this, you can scale to fault-tolerant computation. Requires ~1000 physical qubits per logical qubit.

Fault Tolerance Threshold

Fault-tolerant quantum computing requires: (1) Physical error rate below the threshold; (2) Error correction faster than new errors accumulate; (3) All operations (measurement, gates, preparation) error-corrected. Current superconducting hardware achieves ~0.1–0.5% two-qubit gate errors — approaching but not yet below threshold for large-scale computation.

# Bit-flip error correction code
from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator
from qiskit_aer.noise import NoiseModel, depolarizing_error

# Encode |1⟩ into 3-qubit bit-flip code
def encode_bitflip():
    qc = QuantumCircuit(3)
    qc.x(0)          # Prepare logical |1⟩
    qc.cx(0, 1)      # Spread to all 3 qubits
    qc.cx(0, 2)
    return qc

# Syndrome measurement (detect error without measuring qubit state)
def syndrome_measure(qc):
    # Add 2 ancilla qubits for syndrome
    syndrome = QuantumCircuit(5, 2)
    syndrome.compose(qc, qubits=[0,1,2], inplace=True)
    syndrome.cx(0, 3); syndrome.cx(1, 3)  # Parity of qubits 0,1
    syndrome.cx(1, 4); syndrome.cx(2, 4)  # Parity of qubits 1,2
    syndrome.measure([3, 4], [0, 1])
    return syndrome

# Noise model: 10% bit flip on qubit 1
noise_model = NoiseModel()
error = depolarizing_error(0.1, 1)
noise_model.add_quantum_error(error, ['x'], [1])

# Encode + add noise + measure syndrome
qc = encode_bitflip()
circuit = syndrome_measure(qc)

sim = AerSimulator(noise_model=noise_model)
result = sim.run(transpile(circuit, sim), shots=1024).result()
counts = result.get_counts()
print("Syndrome counts:", counts)
# Syndromes: "00"=no error, "11"=qubit 0 error, "01"=qubit 1 error, "10"=qubit 2 error
