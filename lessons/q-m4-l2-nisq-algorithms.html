NISQ Algorithms & Mitigation

⏱ 45 min 📚 Quantum Computing for Devs 🧩 Lesson 4.2

Core Concepts

VQE (Variational Quantum Eigensolver)

Finds the ground state energy of a molecule. Prepares a parameterized trial state (ansatz), measures its energy expectation, and classically optimizes parameters to minimize it. Used for drug discovery (simulating molecular interactions) and materials science. First quantum advantage claim on real hardware.

QAOA (Quantum Approximate Optimization)

Solves combinatorial optimization problems (MaxCut, traveling salesman, portfolio optimization) approximately. Alternates between problem Hamiltonian and mixer Hamiltonian layers. p layers (circuit depth) trade computation for approximation quality. Competitive with classical solvers for small p on current hardware.

Zero-Noise Extrapolation (ZNE)

Intentionally amplify noise by stretching gate times (2x, 3x), measure results at each noise level, then extrapolate back to zero noise. Implemented in Mitiq. Adds circuit executions but requires no hardware modifications. Effective for shallow circuits with moderate noise.

Probabilistic Error Cancellation (PEC)

Represent each noisy gate as a quasi-probability distribution over ideal gates. Sample from this distribution and average results with appropriate signs to cancel noise. Unbiased estimator of ideal results. Exponential sampling overhead in circuit depth — practical only for shallow circuits.

# VQE for H2 molecule ground state energy
from qiskit_nature.second_q.drivers import PySCFDriver
from qiskit_nature.second_q.mappers import JordanWignerMapper
from qiskit_algorithms import VQE, NumPyMinimumEigensolver
from qiskit_algorithms.optimizers import COBYLA
from qiskit.circuit.library import TwoLocal
from qiskit_aer.primitives import Estimator

# Build H2 Hamiltonian
driver = PySCFDriver(atom="H .0 .0 .0; H .0 .0 0.735", basis="sto3g")
problem = driver.run()
mapper = JordanWignerMapper()
hamiltonian = mapper.map(problem.second_q_ops()[0])

# Classical exact solution (for comparison)
numpy_solver = NumPyMinimumEigensolver()
exact_result = numpy_solver.compute_minimum_eigenvalue(hamiltonian)
print(f"Exact ground energy: {exact_result.eigenvalue.real:.6f} Ha")

# VQE with hardware-efficient ansatz
ansatz = TwoLocal(rotation_blocks="ry", entanglement_blocks="cx",
                  num_qubits=4, reps=2)
optimizer = COBYLA(maxiter=300)
estimator = Estimator()

vqe = VQE(estimator=estimator, ansatz=ansatz, optimizer=optimizer)
vqe_result = vqe.compute_minimum_eigenvalue(hamiltonian)
print(f"VQE ground energy:   {vqe_result.eigenvalue.real:.6f} Ha")
print(f"Error: {abs(vqe_result.eigenvalue.real - exact_result.eigenvalue.real)*1000:.2f} mHa")
