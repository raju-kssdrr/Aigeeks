Amazon Braket & Azure Quantum

Core Concepts

Amazon Braket

AWS quantum computing service giving access to IQM (superconducting), IonQ (trapped ion), QuEra (neutral atom), and Rigetti devices. Pay per task/shot — no commitment. Braket SDK is OpenQASM-based with Python interface. Local simulator (braket_local) is free and fast for testing.

Azure Quantum

Microsoft's platform providing access to IonQ, Quantinuum (trapped ion — highest fidelity), and Rigetti. Quantinuum H2 achieves 99.9% two-qubit gate fidelity — best available. Azure credits available for research. Q# language for native quantum programming; Qiskit/Cirq also supported.

Device Comparison

Superconducting (IBM, Rigetti, IQM): fast gates (~50ns), lower fidelity, short coherence. Trapped ion (IonQ, Quantinuum): slow gates (~100μs), very high fidelity (99.9%), all-to-all connectivity. Neutral atom (QuEra): large qubit counts (256+), programmable connectivity, longer coherence. Match device type to algorithm requirements.

Hybrid Classical-Quantum Jobs

Braket Hybrid Jobs and Azure Quantum Hybrid run classical code co-located with quantum hardware — eliminates network latency between classical optimizer and quantum circuit evaluations. Essential for VQE/QAOA where thousands of circuit evaluations iterate in a tight classical-quantum loop.

# Amazon Braket example
import boto3
from braket.aws import AwsDevice, AwsQuantumTask
from braket.circuits import Circuit

# Create Bell state circuit
circuit = Circuit()
circuit.h(0).cnot(0, 1).measure(0).measure(1)

# List available devices
braket = boto3.client("braket", region_name="us-east-1")
devices = AwsDevice.get_devices(types=["QPU"], statuses=["ONLINE"])
for d in devices:
    print(f"{d.name}: {d.properties.paradigm.qubitCount} qubits")

# Run on IonQ simulator (cheapest)
device = AwsDevice("arn:aws:braket:::device/quantum-simulator/amazon/sv1")
task = device.run(circuit, shots=1000)
result = task.result()
print("Bell state counts:", result.measurement_counts)

# Cost estimation
# QPU costs: IonQ ~$0.01/shot, Rigetti ~$0.0035/shot, Quantinuum ~$0.00065/shot
# Simulator: SV1 ~$0.075/task + $0.00035/shot
